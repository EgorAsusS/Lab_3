# Lab_3

Тема - Булев вектор произвольной длины.

Реализовать все предложенные операции для булева вектора произвольной длины.

Способ представление булева вектора в памяти обсуждалось на занятии. Уметь его рассказать.

Продемонстрировать работу всех операций.

Во многих функциях представлены следующие входные аргументы.

 usigned char *vecA – указатель на массив;

 size_t bits – длина булева вектора(количество разрядов);

Во многих функциях передается именно длина вектора, а не количество ячеек. Количество ячеек, для организации цикла,

можно определять непосредственно в функции по известной вам формуле.

bytes  =  ((bits- 1) / 8) + 1;//вычисление количества ячеек булева вектора(количества байт необходимы для хранения булева вектора длины bits)

 1. Логическое умножение;

unsigned char * logMul(unsigned char *vecA, size_t bitsA, unsigned char *vecB, size_t bitsB);

Функция логически умножает два длинных булева вектора, записывае результат в новый длинный булев вектор, и возвращает указатель на него.
Умножение происходит так: соответсвующие байты векторов умножаются побитовой операцией и записываются в результат. Умножаются векторы одинаковой длины. Размер нового, равен размеру любого из исходных

2. Логическое сложение;

unsigned char * logSum(unsigned char *vecA, size_t bitsA, unsigned char *vecB, size_t bitsB);

Функция логически складывет два длинных булева вектора, записывае результат в новый длинный булев вектор, и возвращает указатель на него.
Сложение происходит так: соответсвующие байты векторов складываются побитовой операцией и записываются в результат. Складываются векторы одинаковой длины. Размер нового, равен размеру любого из исходных

3. Сумма по модулю 2;

unsigned char * sumMod2(unsignedchar *vecA, size_t bitsA, unsigned char *vecB, size_t bitsB);

Функция логически складывет по модулю 2 два длинных булева вектора, записывае результат в новый длинный булев вектор, и возвращает указатель на него.
Сложение по модулю 2 происходит так: соответсвующие байты векторов складываются по модулю 2 побитовой операцией и записываются в результат. Складываются по модулю 2 векторы одинаковой длины.
Размер нового, равен размеру любого из исходных

 4. Инверсия;

void inversion(unsigned char *vec, size_t bits);

Функция инвертрует исходный длинный булев вектор, а также чистит его хвост

5. Сдвиг влево на K разрядов;

void shiftLeft(unsigned char *vec, size_t bits, size_t k);

Функция сдвигает влево исходный длинный булев вектор, а также чистит его хвост. Сдвиг происходит побайтово (если есть сдвиг на целое число байт), а потом на оставшееся число бит (остаток от деления на 8 бит исходного сдвига k).

6. Сдвиг вправо на K разрядов;

void shiftRight(unsigned char *vec, size_t bits, size_t k);

Функция сдвигает вправо исходный длинный булев вектор, а также чистит его хвост. Сдвиг происходит побайтово (если есть сдвиг на целое число байт), а потом на оставшееся число бит (остаток от деления на 8 бит исходного сдвига k).

 7. Установка/сброс К-ого разряда;

void set1(unsigned char *vec, size_t bits, size_t bit);

Функция устанавливает в k разряд 1 (считая слева).

void set0(unsigned char *vec, size_t bits, size_t bit);

Функция устанавливает в k разряд 0 (считая слева).

8. Функция конвертации строки в булев вектор;

unsigned char* convertStrToLongBv(char* str, size_t* bits);

Функция конвертирует строку в длинный булев вектор и возвращает указатель на него. Также функция меняет значение переменной, находящейся по адресу, хранящемуся в указателе bits, на длину булева вектора.

9. Функция конвертации булева вектора в строку;

char* convertLongBvToStr(unsigned char* vec, size_t bits);

Функция конвертирует длинный булев вектор в строку и возвращает указатель на нее. В конце строки ставится символ конца строк '\0'.

10.Функция печати булева вектора на консоль;

void printLongBv(unsigned char* vec, size_t bits);

Функция печатает длинный булев вектор в консоль, последовательно (двумя вложенными циклами) проходясь по каждому биту данного вектора. В конце печатается символ переноса строки '\n'.
